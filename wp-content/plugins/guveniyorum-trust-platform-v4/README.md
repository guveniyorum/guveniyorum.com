# Güveniyorum Trust Platform V4

Mevcut Güveniyorum.com projesini bozmadan koyu temalı premium trust-platform katmanı eklemek için hazırlanmıştır.

## Modüller

- Sol sidebar ve üst arama barı
- Site Ligi
- Şikayet Oluştur
- Marka Yönetimi
- Sertifikasyon
- Oyuncu Psikolojisi ve risk testi
- AI Danışman arayüzü
- Topluluk Merkezi
- Wellness Merkezi
- Kara Liste
- Mobil drawer sidebar

## Kurulum

WordPress yönetim panelinden plugin olarak yüklenebilir. Alternatif olarak WordPress staging ortamında manuel deploy edilir.
